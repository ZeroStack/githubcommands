Common Github Comands

To clone a repostory to your desktop

  1. In the working directory: git clone remote repository URL
  2. Set the repository url to your clone: git remote set-url origin clone remote repository URL

To start pushing changes to your remote repository:

  1. Add files in your local repository: git add .
  2. Commit the files that you have staged in your local repository: git commit -m 'First commit'
  3. Push the changes to Github git push origin master

Reference: https://help.github.com/articles/adding-an-existing-project-to-github-using-the-command-line/
